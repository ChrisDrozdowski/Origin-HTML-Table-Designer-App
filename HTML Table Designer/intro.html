How to use this App

Just a few easy steps to better looking Origin field code generated HTML tables:

  1. In the dialog, define styles or select from a list of Saved Styles (there are already a few saved ones).
  2. Watch the live preview.
  3. Activate the HTML report you want to apply the styles to.
  4. Click the Apply to Report button to apply the current styles to the report.
  5. Optionally Save the styles for future use.

Additionally...

  • The last applied styles are saved as Last Used.
  • Hover mouse over controls to see a popup description of what they do.
  • Share styles with colleagues! Simply export the styles to a CSS file via the Export button. Colleagues can import the file quickly by dragging and dropping it right into the App dialog or using the Import button. They can then save the style to the name of their choosing.
  • To apply the styles to tables you manually create, add the class origin-table to the <table> elements you create (e.g. <table class="origin-table">).
