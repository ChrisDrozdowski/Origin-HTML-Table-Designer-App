How to use this App

Just a few easy steps to better looking Origin-generated HTML tables:

  1. In the dialog, define styles or select from a list of Saved Styles (there are already a few saved ones). Watch the live preview.
  2. Click the Copy to Clipboard button and specify whether the destination is Markup syntax.
  3. Activate your actual report source code and paste the style content.
    • Markup syntax: Paste at the very top of the source code.
    • HTML syntax: Paste just before the end of the <head> element or before any table elements are defined if you have no <head> element.
    • If you are modifying previously pasted styles, be sure to completely replace the old ones.
  4. Optionally Save the styles for future use.
  • Hover mouse over controls to see a popup description of what they do.
  • To apply the styles to tables you manually create, add the class origin-table to the <table> elements you create (e.g. <table class="origin-table">).
  • Share styles with colleagues! Simply export the styles to a CSS file via the Export button. Colleagues can import the file quickly by dragging and dropping it right into the App dialog or using the Import button. They can then save the style to the name of their choosing.
